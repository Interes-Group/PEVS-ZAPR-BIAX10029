Napíšte program, zdrojový kód, v jazyku C použitím štandardu C17, ktorý od používateľa vyžiada hodnotu parameter _n_
a vypíše na obrazovku mriežku s _n riadkami_ a _n stĺpcami_ podľa diagramov nižšie.

```text
+ - - - - + - - - - +
|         |         |
|         |         |
|         |         |
|         |         |
+ - - - - + - - - - +
|         |         |
|         |         |
|         |         |
|         |         |
+ - - - - + - - - - +
```
